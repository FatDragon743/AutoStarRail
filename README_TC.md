## 註意事項

1. 識圖為識別遊戲畫面，遊戲不能被任何視窗覆蓋

2. 不要修改項目根目錄名稱，項目搜索路徑是嚮上搜索項目名稱，若修改路徑則導致項目陷入死循環等。。

3. 自定義體力支援 **擬造花萼**，**凝滯虛影**，**侵蝕隧道**，**歴戰余響**

4. 清理體力需要角色處於**主界面**(可以看到左上角地圖，左上角任務，右邊角色欄等)

5. 運行清理體力該功能後盡量不要操作了，斷點恢復能力不是很好

6. 暫時不支援副本中死亡繼續運行腳本，若中途死亡需要重新運行腳本

7. 路徑中不要有中文路徑！！！
